# humidity-detection-aeco
Object detection of building moisture pathologies using YOLOv8

This project is an experimental tool for detecting moisture.

The results are not infallible and may produce false positives or negatives.

It should not be used as the sole criterion for critical decisions, legal inspections, or structural safety assessments.

The model does not replace professional evaluation.

It is recommended to manually validate the results before taking any corrective action.

Use at the user's own risk.
